# Claude Engineering Framework
## Fase 2 — Skills

Leia completamente toda a estrutura criada na Fase 1.

Nada pode contradizer CLAUDE.md.

Agora crie a pasta

.claude/skills

Cada skill deverá possuir exatamente esta estrutura:

- Objetivo
- Quando utilizar
- Responsabilidades
- Limites
- O que revisar
- O que nunca fazer
- Checklist
- Critérios de aprovação
- Exemplos de atuação

Crie as seguintes Skills:

Business Architect

Software Architect

Supabase Specialist

Vanilla JS Engineer

PWA Specialist

Mobile UX Specialist

Security Reviewer

PostgreSQL Performance

Code Reviewer

QA Engineer

Business Rules Auditor

Documentation Engineer

Product Owner

Technical Critic

Cada especialista deve possuir responsabilidades claras.

Não deve existir sobreposição.

Cada especialista deve complementar os demais.

Ao terminar:

Revise todas as skills.

Verifique redundâncias.

Corrija conflitos.

Atualize README caso necessário.

Pare.
