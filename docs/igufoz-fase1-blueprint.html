📋
Solicitada
Sistema automático
  • Cliente enviou o formulário
  • Salvo no banco
  • WhatsApp automático ao cliente: "Recebemos sua solicitação"
  • Alerta na central via WhatsApp
🔍
Em análise
Central / Admin
  • Admin abre a reserva
  • Verifica disponibilidade manualmente
  • Pode contatar cliente se necessário
  • Decide: confirmar ou recusar
✅
Reserva confirmada
Admin confirma
  • Admin clica "Confirmar"
  • WhatsApp automático ao cliente: confirmação com detalhes
  • Cliente vê status atualizado na área dele
  • Translado liberado para solicitação
🏁
Concluída
Admin após devolução
  • Veículo devolvido
  • Admin marca como concluída
  • Entra no histórico do cliente
  • Base para CRM futuro
CANCELADA Estado especial — não entra no fluxo principal
Pode acontecer em qualquer momento antes de Reserva confirmada. Quem pode cancelar: admin (recusa) ou cliente (desistência via área do cliente). Reserva cancelada fica no histórico — nunca some. Status final imutável.
✈️ Fluxo do Translado
Reserva confirmada Translado fica disponível na área do cliente
→
Cliente solicita Informa voo, data, hora do pouso e nº de pessoas
→
Central recebe alerta WhatsApp + destaque no painel admin
→
Admin confirma Clica "Confirmar translado" no painel
→
Cliente recebe confirmação WhatsApp automático com detalhes do translado
⚠️ Translado só pode ser solicitado quando a reserva está em status Reserva confirmada. Antes disso, o botão não aparece.
